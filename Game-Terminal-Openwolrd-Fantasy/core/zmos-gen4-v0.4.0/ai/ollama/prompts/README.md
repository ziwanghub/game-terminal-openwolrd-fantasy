# Ollama Prompt Layer

`ai/ollama/prompts/` stores prompt templates and prompt fragments prepared for Ollama.
This folder exists to support consistent, governance-aligned prompting without embedding runtime behavior into the repository.

## Intended Contents

- reusable prompt templates
- controlled instruction blocks
- task-specific prompt drafts
- prompt assets for analysis and reporting

## Governance Rule

Prompt files are support assets only.
They do not authorize actions, redefine policy, or override canonical Z-MOS documentation.

Prompt usage must remain consistent with `ai/ollama/policy.md` and `docs/zmos/laws/OLLAMA-LAW.md`.
