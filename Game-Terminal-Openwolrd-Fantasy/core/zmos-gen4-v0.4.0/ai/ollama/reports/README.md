# Ollama Reports Layer

`ai/ollama/reports/` stores generated reports produced for local review.
This folder is intended for advisory outputs such as summaries, assessments, and non-canonical analysis artifacts.

## Intended Contents

- repository summaries
- context-based assessments
- local analysis reports
- draft advisory outputs for human or Codex review

## Governance Rule

Reports in this folder are derived outputs only.
They do not become official truth unless an authorized actor promotes relevant content into canonical project locations.

When evidence is incomplete, reports must not guess and should use:

`not enough evidence`
