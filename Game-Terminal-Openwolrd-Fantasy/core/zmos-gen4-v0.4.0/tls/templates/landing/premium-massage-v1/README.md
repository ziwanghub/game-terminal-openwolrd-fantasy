# premium-massage-v1

Premium landing page template for massage, spa, and beauty businesses.

## Type
`landing`

## Package Support
`pro`, `premium`

## Sections Included
- Hero (CORE)
- Services list (CORE)
- Staff/Therapist (FEATURE — toggleable)
- Gallery (FEATURE — toggleable)
- Booking CTA (FEATURE — toggleable)
- Map/Location (CORE)
- Footer (CORE)

## Required Modules
- `auth-line-v1` — LINE OAuth for customer identification

## Compatible With
- `booking-standard-v1` — Full booking flow
- `admin-spa-v1` — Admin dashboard

## Status
`draft` — Scaffold only, awaiting extraction from TheGig Beauty project

## Usage
1. Copy this template directory to your project
2. Override `config.default.json` with real `shop-config.json`
3. Place source code in `src/`
4. Validate against `config.schema.json` before deploy
