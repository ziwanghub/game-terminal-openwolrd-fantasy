# เศรษฐกิจไอเทม: ประเภท · แรงก์ · ร้าน · ดรอป → ตลาด

**เกม:** `1.14.0-alpha` · **อัปเดต:** 2026-07-14  
**หลัก:** ร้านระบบ = ของทั่วไป · ของแรงก์สูง = ดรอป open world → ตลาดผู้เล่น (ตั้งราคา · ภาษี)

---

## 1. แบ่งตามประเภท (category)

| ประเภท | ตัวอย่าง | ใช้ที่ |
|--------|----------|--------|
| **อุปกรณ์** | ดาบ เกราะ เครื่องประดับ | สวม · ตลาด |
| **ยา/รักษา** | ยา HP ยาพิษ บัฟ | ใช้ทันที |
| **อาหาร** | ขนมปัง เสบียง สตูว์ | needs หิว/ขวัญ (≠ ยา) |
| **หีบ** | sealed_chest_* | เปิดจากกระเป๋าหมวดหีบ |
| **วัตถุดิบ** | upgrade_mat rare_mat | อัป/คราฟ/ขาย |
| **การ์ด** | card_fire … | ใส่ช่องเกียร์ (ตลาดการ์ดยังปิด) |

## 2. แบ่งตามระดับ rarity (เกรดชิ้น)

จาก `data/rarity/tiers.yaml` — ใช้ทั้งดรอป ร้าน ตลาด:

| rank | id | ชื่อ soft | ภาษีตลาดโดยประมาณ* |
|-----:|----|-----------|---------------------:|
| 1 | common | ○ ธรรมดา | ~5% |
| 2 | uncommon | ◇ สูง | ~7% |
| 3 | rare | ◆ หายาก | ~9% |
| 4 | sacred | ★ ศักดิ์สิทธิ์ | ~11% |
| 5 | legendary | ✦ ตำนาน | ~14% |
| 6 | divine | ✧ เทพ | ~16% |
| 7–8 | archdivine / mythic | มหาเทพ / ปฐมกาล | ~สูงสุด ≤18% |

\* ค่าธรรมเนียมหักจากยอดขาย → **tax_fund** (งบกระดานภารกิจ)  
**ราคาตั้งสูง → ภาษีเป็นตัวเงินเยอะขึ้น** (อัตรา % × ราคาที่ตั้ง)

**หีบ rank** (common…SSS/unit) คนละแกนกับ rarity ชิ้น — หีบสูงแค่ตารางดีขึ้น ไม่การันตี legendary

**Unit unique:** save / world — ห้ามร้านระบบ

## 3. แบ่งตาม “ซื้อในร้านระบบได้ไหม”

| ชั้นร้าน | บทบาท | ขายอะไร |
|----------|--------|---------|
| **พ่อค้าเร่** | ของใช้ทุกวัน | ยา อาหาร เกียร์ common วัสดุเล็ก การ์ดพื้นฐาน |
| **ร้านอาวุธเมือง** | เกียร์ต้น–กลาง | common–uncommon (+ การ์ดกลาง) |
| **ตลาดสวรรค์** | sink เงินสวรรค์ | blessed / ธูป / อาหารดี — **ไม่ขายเกียร์ตำนาน** |
| **ตลาดนรก** | sink เงินนรก | hell_contract / void_ash — **ไม่ขายเกียร์ตำนาน** |
| **ตลาดวัสดุหายาก** | วัสดุ/ยาพิเศษ | rare_mat ระดับกลาง — **ไม่ขายเกียร์ rare+** |
| **ศาลาตำนาน** | รับซื้อแรงก์สูง | **stock ระบบว่าง** — ของตำนานมาจากดรอปเท่านั้น |

### นโยบาย (ล็อก)

1. ร้านระบบ **ไม่มี** thorn / shadow rare+ / unit / sealed chest / void_key  
2. ของแรงก์สูง = **ดรอปโลก · บอส · หีบ · เควส**  
3. ผู้เล่นเอาของนั้นไป **ตลาดกลาง (M)** ตั้งราคาเอง

## 4. ดรอป → ตลาดผู้เล่น → ภาษี

```text
  open world / บอส / หีบ / เควส
           │
           ▼
     ได้ไอเทม (มี rarity ชิ้น)
           │
           ▼
   ตลาดกลาง · ฝากขาย · ตั้งราคา
           │
           ├─ ผู้ซื้อจ่ายเต็มราคา
           ├─ ผู้ขายได้หลังหักค่าธรรมเนียม
           └─ ภาษี → tax_fund → ค่าจ้างกระดาน J
```

| ความจริงในเกม | รายละเอียด |
|---------------|------------|
| ตั้งราคาเอง | ได้ ภายใน ~1/5–5× ราคาแนะนำ |
| ราคาสูง | ภาษี (ตัวเงิน) สูงตาม % × ราคา |
| rarity สูง | % ภาษีสูงขึ้นเล็กน้อย |
| การ์ด | ฝากขายตลาดยังไม่เปิด (เฟสถัดไป) |
| unit unique | ฝากขายได้แต่ของชิ้นเดียว — ระวัง |

## 7. เกณฑ์คุณภาพเศรษฐกิจ

1. ผู้เล่นใหม่ซื้อยา/อาหาร/ดาบต้นได้ทันทีจากร้าน  
2. ของ “เท่/แรง” ต้องออกไปเดินโลก — ไม่ซื้อตรงจาก NPC ตำนาน  
3. ตลาดผู้เล่นมีของหายากหมุน — ระบบกินภาษีไปเลี้ยงกระดาน  
4. Soft / anti-spoiler — ไม่โชว์ตาราง % ภาษีดิบใน UI แค่ “แนะนำราคา” + ยอดกองทุน soft
