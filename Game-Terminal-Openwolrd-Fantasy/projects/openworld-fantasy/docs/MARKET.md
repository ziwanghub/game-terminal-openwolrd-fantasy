# ตลาดกลาง (Player Market)

**สถานะ:** 1.13.0-alpha  
**ไฟล์:** `saves/{world}/market.json` แชร์ทั้งโลก

---

## หลักการ

1. ผู้เล่น **ฝากขาย** ไอเทมจากคลัง → ขึ้นตลาดกลาง  
2. ผู้เล่นอื่น (เซฟคนละตัวในโลกเดียวกัน) **ซื้อ** ได้  
3. เมื่อซื้อสำเร็จ:
   - ผู้ซื้อเสียเงินเต็มราคา · ได้ของ  
   - ผู้ขายได้เงินหลังหัก **ค่าธรรมเนียมตลาด**  
   - ผู้ขายได้ **รายงาน** ว่าใครซื้อ (inbox / ตอนเข้าตลาดหรือโหลดเซฟ)

---

## เศรษฐศาสตร์ soft

| กลไก | ผล |
|------|-----|
| ราคาแนะนำ | จากราคาสินค้า + rarity + ค่าเฉลี่ยขายเก่า |
| อุปทาน | รายการเดียวกันหลายชิ้น → แนะนำราคาลง |
| ค่าธรรมเนียม / **ภาษี** | หักจากยอดผู้ขาย (~5–18% ตาม rarity) → เข้า **tax_fund** |
| กองทุนภาษี | ใช้เป็น **ค่าจ้างประกาศภารกิจ** (กระดาน J) |
| เพดานราคา | ห้ามตั้งสูง/ต่ำเกิน ~5× ราคาแนะนำ |

ผู้เล่นไม่เห็นสูตรเต็ม — เห็นแค่ “แนะนำราคา ~X” · กองทุนภาษีแสดงยอดรวม soft

## UI

กระเป๋า (5) → **M ตลาดกลาง**

```text
 1. ดู/ซื้อจากตลาด
 2. ฝากขายของจากคลัง
 3. ถอนของที่ตัวเองฝากขาย
 4. รายงานการขายล่าสุด
 0. กลับ
```
