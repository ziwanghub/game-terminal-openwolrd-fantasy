| **WO** | WO-048 · **WO-049 surface** |
| **สถานะ** | locked · surface shipped @ 1.96 |

---

## 1. สองชั้นเกรด (ห้ามปน)

| ชั้น | รหัส | ความหมาย | ผล |
|------|------|----------|-----|
| **เกรดรวม** | `player_grade` | ระดับ “คน / เสี้ยวพร” F–SSS | อัตราเติบโต · โปรไฟล์สมดุล · (อนาคต) ผันผวน |
| **เกรดแกน** | `axis_grades[atk/defense/magic/speed]` | โจมตี/กัน/เวท/เร็ว | โชว์ใน P/V หลังปลดวิหาร |

**Anima (จิตวิญญาณ)** แยกชั้น — ไม่ใช่เกรดโจมตี · ไม่ปนขวัญ

---

## 2. ตารางเกรดแกน (ล็อก · จาก score ซ่อน)

`axis_score` ซ่อน = `alloc[axis] * growth_mult + soft_base`  
(ไม่โชว์ผู้เล่น)

| เกรด | คะแนนแกน (ซ่อน) | Soft label (ตัวอย่าง) |
|------|-----------------|------------------------|
| **F** | 0–4 | นักฝึก · ยังไม่ชัด |
| **E** | 5–9 | อดทนได้ · พอใช้ |
| **D** | 10–14 | เริ่มมีแวว |
| **C** | 15–19 | จิตต้น · มีศักยภาพ |
| **B** | 20–26 | โดดเด่น · ไร้เงา |
| **A** | 27–36 | ยอดเยี่ยม · คมชัด |
| **S** | 37–46 | ตำนานแผ่ว · เหนือมนุษย์แผ่ว |
| **SS** | 47–56 | ใกล้เทพ |
| **SSS** | 57+ | เสี้ยวเทพ · คุมไม่นิ่ง |

> ขอบไม่ทับ · ขั้นต้น/กลาง/ปลาย/พิเศษ = **WO-049** (soft surface · ยังไม่ appraisal ละเอียด)

---

## 2.1 Soft Tier ภายในเกรด (WO-049 ล็อก)

ภายในช่วงคะแนนของตัวอักษรเดียวกัน แบ่ง 4 ชั้น (ซ่อน score · โชว์คำ soft เท่านั้น):

| tier รหัส | Soft label | Soft flavor |
|-----------|------------|-------------|
| `early` | ขั้นต้น | เริ่ม |
| `mid` | ขั้นกลาง | มั่น |
| `late` | ขั้นปลาย | เต็มช่วง |
| `special` | พิเศษ | เหนือขอบ |

| กฎ (ซ่อน) | |
|-----------|--|
| แบนด์ปิด (F–SS) | แบ่งตามตำแหน่งในแบนด์ ≈ 0–33% / 33–66% / 66–92% / 92%+ = special |
| SSS เปิด | ต้น 57–66 · กลาง 67–79 · ปลาย 80–99 · พิเศษ 100+ |

**ตัวอย่าง surface:** `โจมตี (นักรบฝึกหัด) C · ขั้นต้น`

## 4. growth_profile (ล็อก)

สุ่ม/กำหนดตอนปลดวิหาร (จากสไตล์เล่น latent — ช่วง 048 ใช้ seed จาก alloc+level)

| profile | ผลต่อแกน (ตัวคูณทิศทาง) |
|---------|-------------------------|
| **balanced** | ทั้ง 4 แกน ×1.00 |
| **focused** | แกนสูงสุด ×1.25 · ที่เหลือ ×0.75 |
| **mixed** | 2 แกนสูงสุด ×1.15 · ที่เหลือ ×0.85 |

`effective_growth(axis) = growth_mult(player_grade) * profile_tilt(axis)`

---

## 5. วิหารปลดพลัง (Temple Unlock)

| เงื่อนไข | |
|----------|--|
| **ต้อง** | `level >= 10` **และ** soft flag “ตัน” |
| Soft flag “ตัน” | ยังไม่ `grade_revealed` และ (stat_points≥5 **หรือ** level≥10 กับ kills≥8 **หรือ** `_grade_pressure` จากเล่น) |
| **ผล** | `grade_revealed=True` · สุ่ม/คำนวณ `player_grade` + `growth_profile` · เปิดโชว์เกรดแกน |
| **เปิดเผย 048** | ตัวอักษรเดียว + soft description (ไม่โชว์ขั้นต้น/ปลาย) |

สถานที่: พื้นที่ `ancient_city` (เมนูสนาม / ตัวละครเมื่อเงื่อนไขถึง) · soft ใบ้ได้ทุกที่เมื่อถึงเกต

## 7. เมนู P (หลังปลด · WO-049)

```
ลงแต้มสถานะ (แต้มคงเหลือ: N)

ระดับคุณ 〔B〕 โดดเด่น · สมดุล
(เกรดแกน + ชั้น soft · ไม่โชว์ตัวเลข)

1. โจมตี (นักรบฝึกหัด) C · ขั้นต้น  ·  รู้สึกมือเริ่มหนักขึ้น
2. ป้องกัน (อดทนได้) E · ขั้นกลาง
3. เวท (จิตต้น) E · ขั้นต้น
4. ความเร็ว (ไร้เงา) B · ขั้นปลาย

0 = กลับ
```

ก่อนปลด: ไม่มีตัวอักษรเกรด / tier — soft feel อย่างเดียว
