# ระบบบรรยายสถานการณ์ (Combat Narrative)

สร้างอารมณ์ให้อ่านรู้ว่า**กำลังเกิดอะไรขึ้น** ระหว่างไฟต์/สถานะ

## สิ่งที่บรรยาย
| เหตุการณ์ | ตัวอย่าง |
|-----------|----------|
| เปิดไฟต์ / ซุ่ม / บอส | อากาศเปลี่ยน · แรงกดดัน |
| ใช้สกิล / คอมโบ / ฮีล | ปล่อยท่า · ลูกโซ่ · แสงฟื้น |
| ดาเมจออก (แรงตาม %) | ครูด / กระแทก / พายุ |
| ดาเมจเข้า + ผลกัน | กันถูก/ผิด · เจ็บแปลบ/เกือบขาดใจ |
| ธาตุ | ไฟ น้ำ ฟ้า เงา ศักดิ์สิทธิ์ น้ำแข็ง |
| สถานะ (บัพ/ดีบัพ) | พิษไหม้แช่แข็ง · หมดอายุ |
| ปาร์ตี้เรียก / ช่วยตี | เข้าแนว · ช่วยโจมตี |
| หนี / ชนะ / แพ้ | บรรยากาศปิดฉาก |
| แถบสถานการณ์ | `▸ คุณยังมั่น · ศัตรูสะเทือน · บัฟ:...` |

## ไฟล์
- `data/narrative/combat_flavor.yaml` — ข้อความสุ่มได้
- `game/domain/narrative.py` — เลือกบรรทัด + จัดรูปแบบ
- เชื่อมใน `field_loop` คอมแบต

## ตัวเลข
ยังแสดง `[ตัวเลข] ดาเมจ N` แยกจากบรรยาย — อ่านง่ายทั้งอารมณ์และข้อมูล

## Field narrative (1.6.2)
ไฟล์ `data/narrative/field_flavor.yaml` รวมกับ combat:

| เหตุการณ์ | key |
|-----------|-----|
| พัก / สำรวจ / เดินทาง | field_rest, field_explore*, field_travel* |
| เป้าในพื้นที่ | field_sights / field_no_sights |
| เข้าหา NPC/มอน/ผู้เล่น/หีบ | field_approach_* |
| ดรอป | field_loot* |
| ห้องสมุด | field_library / denied |
| ปาร์ตี้ | field_party_* |
| ผล NPC | npc_outcome_* |
| อารมณ์พื้นที่ | area_mood_<area_id> |
| บัฟหมด | blessing_expire |

Registry โหลดทุกไฟล์ใน `data/narrative/*.yaml` แล้ว merge

## ขยาย
เพิ่ม key ใน YAML ได้เลย engine จะสุ่มใช้  
ปิดทั้งระบบ: `enabled: false` ใน combat_flavor.yaml
