## 1. เป้าหมาย

ออกแบบ **จังหวะอ่าน–ตัดสินใจ–กด** ให้:

1. **รู้สถานะตัวเองใน 1 วินาที** (ยังมีชีวิตไหม / พลังพอไหม / อยู่ไหน)
2. **รู้ว่าทำอะไรได้ใน 2 วินาที** (ตัวเลือกชัด ไม่ซ่อนในกล่องยาว)
3. **บรรยายโลกไม่กลบตัวเลขสำคัญ**
4. **เหตุการณ์พิเศษเด่น** โดยไม่ spam ทุกเทิร์น
5. **รองรับมือใหม่ + ผู้เล่นเก๋า** ด้วยโหมดความหนาแน่น (density)

**ไม่ใช่เป้าหมายตอนนี้:** GUI, animation หนัก, responsive web, Z-MOS ควบคุม UI

## 2. หลักการออกแบบ (Text UX Principles)

| # | หลัก | ทำยังไง |
|---|------|---------|
| P1 | **โครงคงที่ (Spatial memory)** | ทุกจอ: สถานะ → บริบท → เรื่อง → เมนู → พรอมต์ |
| P2 | **สำคัญก่อนสวย** | HP/MP/ทางเลือก ก่อน ASCII / flavor |
| P3 | **ปกติเรียบ · พิเศษดัง** | Field เรียบ; บอส/LEVEL UP/ของหายาก = Spotlight |
| P4 | **ไม่สปอยล์สูตร** | soft label, ??? , ไม่โชว์ตารางธาตุเต็ม |
| P5 | **หนึ่งพรอมต์หนึ่งคำถาม** | `เลือก:` เดียวต่อขั้น; ซับเมนูมี `0=กลับ` |
| P6 | **ภาษาไทย UI** | ระบบไทย; ชื่อสกิล/ไอเทมตาม data |
| P7 | **ความกว้างคงที่** | เป้า **64 คอลัมน์** (อ่านมือถือ SSH ได้) · max art 60 |
| P8 | **Density ตามบริบท** | ออโต้ = L0 · สนาม = L1 · ตรวจของ = L1+ · เหตุการณ์ = L2 |
| P9 | **ตัวเลขแยกบรรทัด** | บรรยาย + `[ตัวเลข] ดาเมจ 12` ไม่ปนจนอ่านไม่ออก |
| P10 | **ปิดของรกได้** (prefs ภายหลัง) | ปิด art, ปิด flavor ยาว, ปิด typewriter |

### Anti-patterns (ห้าม)

- กล่องสถานะ 15+ บรรทัด **ทุก** เทิร์นสนาม
- เมนูการกระทำซ้ำสองบล็อก
- ASCII มอนเต็มจอทุกไฟต์
- บังคับ Enter ระหว่างออโต้
- ซ่อน `0=กลับ` / ทางรอด
- โชว์เลเวลมอน / difficulty ตัวเลขดิบก่อน unlock

## 3. โครงจอสากล (Screen Frame)

ทุกโหมดเล่นใช้ **5 โซน** เรียงจากบนลงล่าง:

```text
┌─ A. CHROME ─────────────────────────────────────┐  1 บรรทัด (โหมด · โลก)
├─ B. VITALS ─────────────────────────────────────┤  L0 หรือ L1-compact
├─ C. CONTEXT ────────────────────────────────────┤  ที่ / เป้า / ชั้นดัน / ไฟต์
├─ D. NARRATIVE ──────────────────────────────────┤  0–N บรรทัด flavor (จำกัด)
├─ E. ACTIONS ────────────────────────────────────┤  ตัวเลือก + คีย์ลัด
└─ F. PROMPT ─────────────────────────────────────┘  เลือก: _
```

| โซน | หน้าที่ | ความสูงเป้า |
|-----|---------|------------|
| **A Chrome** | บอกโหมด: สนาม / ไฟต์ / ดัน / ร้าน | 1 |
| **B Vitals** | ตัวละครมีชีวิตไหม | 1 (L0) หรือ 4–7 (L1) |
| **C Context** | โลกภายนอกกำลังเกิดอะไร | 2–8 |
| **D Narrative** | อารมณ์ ไม่บังคับอ่านตัวเลข | 0–4 (cap) |
| **E Actions** | ทำอะไรได้ | 2–6 |
| **F Prompt** | input | 1 |

**กฎทอง:** ถ้าจอสูงเกิน ~28 บรรทัดในเทิร์นปกติ = หนาเกินไป → ตัด D หรือย่อ B

## 4. ระดับการแสดงผล (Density)

| ระดับ | ชื่อ | ใช้เมื่อ | Vitals | Narrative | Art |
|-------|------|---------|--------|-----------|-----|
| **L0** | Compact strip | ออโต้, tick เร็ว, หลังแอคชันสั้น | 1 บรรทัด | ไม่มี/1 บรรทัด | ไม่ |
| **L1c** | Field compact | **เทิร์นสนามปกติ (แนะนำใหม่)** | 4–5 ในกล่อง | 0–2 | ไม่ |
| **L1** | Standard full | เข้าสนามครั้งแรก, กด `S` ดูเต็ม, เมนูย่อยสำคัญ | กล่องเต็มปัจจุบัน | ตามเหตุการณ์ | ไม่ |
| **L1+** | Inspect | ดูไอเทม / journal / เกียร์ละเอียด | +แผงย่อย | flavor ชิ้นนั้น | optional small |
| **L2** | Spotlight | LEVEL UP, บอสเปิด, ของหายาก, เคลียร์ดัน | ย่อชั่วคราว | กล่องเด่น + art | ใช่ |

### L0 strip (มาตรฐานใหม่)

```text
อัศวินเมฆ Lv.12 · HP ████████░░ 82/100 · MP ███░░░░░░░ 28/60 · ป่ามืด
```

### L1c — สนาม (แนะนำแทน L1 เต็มทุกเทิร์น)

```text
╔══════════════════════════════════════════════════════════╗
║  อัศวินเมฆ  Lv.12 นักรบ · ยอดนักดาบ                       ║
║  HP [████████░░] 82/100   MP [███░░░░░░░] 28/60          ║
║  เงิน โลก 340 · สวรรค์ 8 · นรก 3    กดดัน 22             ║
║  ที่ ป่ามืด (ชำนาญ 41%)   ปาร์ตี้ 1/3                     ║
╚══════════════════════════════════════════════════════════╝
```

**ย้ายออกจากทุกเทิร์น** (ไปจอ `S` / `I` ตรวจเต็ม): รายการสกิลยาว, แท็กเกียร์, เซ็ต, ศิษย์, ผู้เล่นอื่น, “เลเวลไม่จำกัด…”

### L1 full — เปิดเมื่อต้องการ

เหมือน `render_status_l1` ปัจจุบัน + จัดกลุ่ม:

1. ตัวตน (ชื่อ เลเวล อาชีพ ราศี unit)  
2. หลอด (XP HP MP กดดัน สถานะ)  
3. เศรษฐกิจ + ที่  
4. เกียร์ย่อ  
5. หมายเหตุ (แต้มค้าง / พร)

## 5. แผนที่จอ + Wireframe

### 5.1 Boot / เมนูหลัก

```text
╔══════════════════════════════════════════════════════════╗
║  Open World Fantasy                                      ║
║  Terminal Text RPG  ·  1.9.x  ·  text-ui                 ║
╠══════════════════════════════════════════════════════════╣
║  โลกเปิด · ความยากหลายระดับ · เรียนรู้จากประสบการณ์      ║
╚══════════════════════════════════════════════════════════╝

  1. เข้าโลก (เล่นต่อ / สร้างใหม่)
  2. โหลดตัวละคร
  3. อันดับชื่อเสียงของโลก
  4. เกี่ยวกับ
  5. นำเข้าเซฟ
  8. แอดมิน
  0. ออก

เลือกหมายเลข: _
```

**UX:** แยก “เล่น” กับ “ระบบ” ด้วยระยะห่าง; เกี่ยวกับไม่บังคับอ่านก่อนเล่น

### 5.2 สร้างตัวละคร (Wizard ทีละขั้น)

ขั้นละ 1 คำถาม — ไม่ยัดฟอร์มยาว:

```text
── สร้างตัวละคร (1/4) ชื่อ ──
ชื่อตัวละคร: _
```

```text
── สร้างตัวละคร (2/4) เพศ ──
เพศ (ชาย/หญิง/อื่นๆ · Enter=ไม่ระบุ): _
```

```text
── สร้างตัวละคร (3/4) วันเกิด ──
วัน/เดือน/ปี (เช่น 15/6/2000): _
→ ราศี: เมถุน  (ไม่โชว์โบนัสตัวเลข)
```

```text
── สร้างตัวละคร (4/4) อาชีพ ──
  1. นักรบ — แนวหน้า ดาเมจกายภาพ
  2. นักเวท — มานาและธาตุ
  ...
เลือก (หมายเลข): _
```

สรุปสั้นก่อนเข้าโลก:

```text
┌─ พร้อมออกเดินทาง ─────────────────┐
│  อัศวินเมฆ · นักรบ · ราศีเมถุน     │
│  HP 120  MP 40  (ค่าตั้งต้น)      │
│  กด Enter เพื่อเริ่ม · 0 แก้ใหม่  │
└───────────────────────────────────┘
```

### 5.3 สนาม (Field) — จอสำคัญที่สุด

```text
〔สนาม〕 โลก default
┌──────────────────────────────────────────────────────────┐
│  อัศวินเมฆ  Lv.12 นักรบ · ยอดนักดาบ                         │
│  HP [████████░░] 82/100  MP [███░░░░░░░] 28/60  กดดัน 22 │
│  เงิน 340/8/3   ป่ามืด · ชำนาญ 41%   ปาร์ตี้ 1/3          │
└──────────────────────────────────────────────────────────┘

── สิ่งที่สังเกต ──
  1. [มอน] เงาเคลื่อนไหวใต้พุ่ม — เสี่ยง:ปานกลาง
  2. [หีบ?] กล่องไม้ผุ กลิ่นโลหะ — เสี่ยง:ต่ำ
  3. [ทาง] ปากโพรงรากไม้เย็นผิดปกติ — เสี่ยง:?

  ลมเย็นพัดใบไม้แห้งเป็นวง...          ← narrative สั้น (ไม่ทุกเทิร์น)

── ทำอะไรต่อ ──
 1 พัก   2 สำรวจ   3 เข้าหา   4 เดินทาง   5 กระเป๋า/ร้าน
 6 ความรู้   7 เซฟ   8 ออโต้   9 เควส
 S สถานะเต็ม  N นิสัย  K สกิล  Y ปาร์ตี้  B บอส
 H ช่วย  T บทเรียน  0 ออก(เซฟ)

เลือก: _
```

**การเปลี่ยนจากของปัจจุบัน**

| เดิม | ใหม่ |
|------|------|
| L1 เต็ม ~15 บรรทัดทุกเทิร์น | **L1c ~4 บรรทัด** |
| เมนู “การกระทำ” ซ้ำ 2 ครั้ง | **บล็อกเดียว** |
| สกิล/แท็กในสถานะสนาม | ย้ายไป `S` |
| mood พื้นที่ %4 | คงไว้ แต่ **1 บรรทัด** ใต้ sights |
| บอสหิ้นต์ยาว | 1 บรรทัดใต้ sights ถ้าปลดแล้ว |

**Sights risk soft labels (ไม่ใช่ตัวเลข)**

| ภายใน | แสดง |
|--------|------|
| ต่ำ | ต่ำ / แผ่ว |
| กลาง | ปานกลาง |
| สูง | สูง / อึด |
| ไม่รู้ | `?` |

### 5.4 ต่อสู้ (Combat)

```text
〔ไฟต์〕 ป่ามืด
── รอบ 3 ──────────────────────────────────────
 คุณ  HP [██████░░░░] 61/100  MP [██░░░░░░░░] 14/60
 ศัตรู ???  HP ???/???   ▸ ท่าทาง: เตรียมพุ่งเข้าหา

  เงากระชากเข้าใกล้ — คุณพลาดจังหวะก้าวถอย
  [ตัวเลข] ดาเมจเข้า 11

── แอคชัน ──
 1 โจมตี   2 สกิล/คอมโบ   3 ยา   4 หนี   5 เรียกปาร์ตี้
   คอมโบสูงสุด 3 ขั้น · มานาไม่พอใช้โซ่ไม่ได้

เลือก: _
```

**เมื่อรู้มอน (journal / ชนะมาก่อน)**

```text
 ศัตรู หมาป่าร้าย  HP [████░░░░░░] 40/90  [ไฟ]  ▸ เฟส -
```

**เฟสกัน (defense)**

```text
  ศัตรู: เปลวไฟรวมตัว!
── กันอย่างไร ──
 1 กายภาพ   2 น้ำ   3 ลม   4 ไมกัน
เลือก: _
```

**กฎไฟต์**

1. แถบ vitals ทุกต้นรอบ (ไม่ใช้กล่องคู่ใหญ่)  
2. `▸ situation` 1 บรรทัดอารมณ์  
3. ผลลัพธ์: flavor แล้วตามด้วย `[ตัวเลข]`  
4. บอสเปิดครั้งแรก = **L2 spotlight** ครั้งเดียว  
5. ชนะ: สรุป XP/เงิน/ดรอปในกล่องสั้น แล้ว loot choice  

**จบไฟต์ (Victory panel)**

```text
┌─ ชนะ ────────────────────────────┐
│  XP +24  (120/200 · 60%)         │
│  เงินโลก +18                     │
│  ชำนาญพื้นที่ +3%                │
└──────────────────────────────────┘
── ของที่พบ (เลือกเก็บ · 0 ข้ามทั้งหมด) ──
  1. ยา HP (common)
  2. วัสดุอัพ · สัมผัส: เย็นแปลก
เก็บหมายเลข (คั่นด้วย , ได้): _
```

### 5.5 ดันเจียน

```text
〔ดันเจียน〕 รากป่ารูปเขี้ยว · ชั้น 2/3
┌──────────────────────────────────────────────────────────┐
│  HP [█████░░░░░] 55/100  MP [██░░░░░░░░] 18/60  กดดัน 41 │
│  สภาพ: ทางคดเคี้ยว · อากาศอับ                              │
│  (ความรู้สึก: เวลาในนี้ไม่เหมือนข้างนอก...)               │  ← soft time เท่านั้น
└──────────────────────────────────────────────────────────┘

  1 สำรวจชั้นนี้   2 ลงลึกกว่านี้   3 หาทางออก/หนี
  4 พักสั้น (เสี่ยง)   5 สถานะ   0 พยายามออก (ถ้าปลดล็อก)

เลือก: _
```

**ห้าม:** โชว์ `difficulty: 3` หรือ `turns left: 4`  
**อนุญาต soft:** “เหมือนจะถูกบีบจากเงา…” เมื่อใกล้หมดเวลา

### 5.6 กระเป๋า / เกียร์ / ร้าน

**โครงเมนูย่อยเดียวกัน**

```text
〔กระเป๋า〕 12/40 ช่อง
  สวม: ดาบเหล็ก (uncommon) · เกราะหนัง (common)

  1 สวมอุปกรณ์   2 การ์ด   3 อัปเกรด   4 ร้าน   5 คราฟ
  6 ดูไอเทม   7 เกียร์ละเอียด   0 กลับสนาม

เลือก: _
```

**ดูไอเทม (L1+)** — ซ่อนสูตรคราฟ/อัป

```text
┌─ ยา HP ขนาดเล็ก ─────────────────┐
│  ประเภท 消耗 · ระดับ common       │
│  สัมผัส: ขวดอุ่นเล็กน้อย           │
│  ใช้แล้ว: ฟื้นกำลังในสนาม/ไฟต์    │
│  (ตัวเลขฮีลไม่โชว์ตรงๆ ถ้า soft)  │
└──────────────────────────────────┘
Enter...
```

**ดูอุปกรณ์ / ดาบ (L1+ → อลังตาม rarity)** — `gear_showcase.py`

รายการ: `iron_sword  ดาบเหล็ก  [Lv.1 ○ธรรมดา]`  
พิมพ์ไอดีหรือหมายเลข → กรอบ + ASCII + อรรถรสตาม **ระดับอาวุธ (rank 1–8)**  
common เรียบ · legendary มีแบนเนอร์ ✦ · mythic กรอบ ◈ สูงสุด (ดู `docs/BAG_SYSTEM.md` §4.3)

```text
┌──────────────────────────────────┐
│ ดาบเหล็ก [○ธรรมดา]               │
│ ไอดี: iron_sword                 │
│ ระดับอาวุธ: Lv.1  [○ธรรมดา]      │
│      /|                          │
│     //|   ← ดาบธรรมดา            │
│  เหล็กเรียบ — ดาบนักเดินทางทั่วไป │
└──────────────────────────────────┘
```

**ร้าน**

```text
〔ร้าน〕 พ่อค้าเร่ · รับซื้อระดับ common–rare
  เงินคุณ  โลก 340 · สวรรค์ 8 · นรก 3

  1. ยา HP        35ว
  2. ดาบเหล็ก ★   120ว   ← rarity soft star ไม่บังคับตัวอักษร EN
  ...
  B ขายของ   F กรองระดับ   0 กลับ

เลือก: _
```

### 5.7 สกิลต้นไม้ / นิสัย / ปาร์ตี้ / เควส

**แพทเทิร์นเดียว:** หัวข้อ → รายการเลข → คำใบ้ต้นทุน soft → ยืนยัน → `0 กลับ`

```text
〔สกิล · นักรบ〕 แต้ม/เงินไม่โชว์สูตรเต็ม
  ปลดแล้ว: ฟาดพื้นฐาน, กันพื้นฐาน, ...
  เรียนได้:
```

## 7. ลำดับชั้นข้อมูล (Information Hierarchy)

```text
P0  Immediate survival     HP, ตาย/สลบ, ล็อกดัน
P1  Decision fuel          MP, ตัวเลือกแอคชัน, สิ่งรอบตัว
P2  Orientation            ที่, ชั้นดัน, ชื่อศัตรู/???
P3  Economy & growth       เงิน, XP%, เควส
P4  Identity & build       อาชีพ, เกียร์, สกิล, นิสัย

```

**กฎ:** P5 ห้ามดัน P0–P1 ออกนอกจอ · P6 ไม่โชว์ทุกเทิร์น

### Soft vs Hard numbers

| แสดง hard ได้ | soft / ซ่อน |
|---------------|-------------|
| HP/MP ตัวเอง | สูตรดาเมจ, ตารางธาตุเต็ม |
| เงิน 3 สกุล | โอกาสคราฟ %, ดรอป table |
| XP% แถบ | XP curve formula |
| ชื่อไอเทม + rarity คำ | difficulty ดันตัวเลข |
| ดาเมจรอบนี้ `[ตัวเลข]` | เลเวลมอน, stat ศัตรูดิบ |

## 8. Feedback & จังหวะ (Pacing)

| เหตุการณ์ | Feedback |
|-----------|----------|
| แอคชันสนามสำเร็จ | 1 บรรทัดผล + กลับ field frame |
| ไฟต์แต่ละจังหวะ | narrative + `[ตัวเลข]` |
| เลเวลอัพ | L2 แล้วค่อย field |
| ตาย soft | กล่องสรุปเสียหาย → ฟื้นที่จุดปลอดภัย |
| เควสคืบ | บรรทัด `เควส: ...` ใต้ผลแอคชัน |
| ผิดพลาด input | `ไม่เข้าใจ — พิมพ์ H ดูช่วย` ไม่ stack trace |

**Empty input:** ถือเป็น “ไม่ทำอะไร / รีเฟรชจอ” หรือเตือนครั้งเดียว — **อย่า** วน tutorial

## 9. Accessibility & เทอร์มินัล

| หัวข้อ | ข้อกำหนด |
|--------|----------|
| ความกว้าง | 64 default (`UI_WIDTH`) · อย่าบังคับ 80+ |
| Unicode box | เปิดได้; fallback ASCII `+ - \|` |
| สี | optional Rich; **อ่านได้ขาวดำ** |
| ภาษา | UI ไทย; error ไทย |
| Screen reader-ish | อย่าพึ่งสีอย่างเดียวบอกอันตราย (ใช้คำว่า “อันตราย”) |
| Thai width | รู้ว่า display width ≠ len(); ระยะสั้นยอมเบี้ยวเล็กน้อย |
| SSH / iPad | L1c + ไม่พึ่งเมาส์ · ไม่ clear screen บังคับ (optional `cls` pref) |

**Clear screen:** default **ไม่เคลียร์** ทั้งจอ (เลื่อนลงอย่างเดียว) — ลดหลอน; optional pref `ui.clear_each_turn`
