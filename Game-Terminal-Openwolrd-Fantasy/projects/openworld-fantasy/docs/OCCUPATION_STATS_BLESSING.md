# อาชีพ · สเตตัส · พร · ฉลาด · โชค · latent · ห้องสมุด · Unit

**สถานะ:** implement (1.12.x+) · รวมเนื้อหาเดิมจาก `OCCUPATION_PROGRESSION`  
**หลัก:** ผู้เล่น**ไม่รู้สูตร** · ไร้ขีดจำกัด · soft label · ตัวเลขใน YAML  

> Doc Policy: ไฟล์นี้เป็น single source ของโดเมนอาชีพ/สเตตัส — อย่าแยก md ซ้ำ

---

## 1. จุดเริ่ม (เมือง · ยังไม่มีอาชีพ)

```text
เข้าโลก → ชื่อ / เพศ / วันเกิด
         → นักเดินทาง (vagabond) — ยังไม่เลือกอาชีพ
         → แต้มเริ่มต้น 4–9 (สูตรซ่อน)
         → เมือง · ภารกิจง่าย
```

| เห็น | ซ่อน |
|------|------|
| แต้มคงเหลือ X | ทำไมได้ X |
| แจกแต้ม 1–5 | ผลแฝงจริง |
| เมนู **C** เมื่อ “ถึงเวลา” | checklist เงื่อนไข |

---

## 2. ค่าสถานะ

### ที่แจกได้ (เมนู **P**)

| UI | คีย์ | ผลแฝง (ไม่โชว์สูตร) |
|----|------|---------------------|
| โจมตี | atk | ดาเมจ + คริแฝง |
| ป้องกัน | defense | ลดดาเมจ + โต HP |
| เวท | magic | เวท + โต MP |
| ความเร็ว | speed | ลำดับ/หลบ + ลดดาเมจโอกาส |
| ความฉลาด | intelligence | ขยายความจุสติ + ทางเลือกพิเศษ |

- เลเวลอัพได้แต้มตามอาชีพ (`points_per_level` ใน YAML · soft)  
- UI โชว์**แต้มลงทุน** ไม่โชว์ตัวคูณ latent  

### โชค (ไม่โชว์ในเมนู)

- ปัจจัยซ่อน: วันเกิด · area · ปาร์ตี้ · เซ็ตเกียร์ · `latent_seed`  
- soft bias ทั้งดี/ร้าย — ไม่ always buff  

### ความฉลาด = ทรัพยากร

| ชั้น | ความหมาย |
|------|----------|
| ลงทุนใน P | ขยาย `intel_max` |
| `intel_current` | ใช้แลก ATB / ทางเลือก ★ |
| ฟื้น | พัก · เวลาสนาม · ชาสมาธิ / ยาน้ำความคิด / ธูปแจ่มจิต |

- ไฟต์ **6** = ใช้สติเร่งแท่งจังหวะ  
- NPC ★ = ใช้สติ · โชคซ่อนอัตราสำเร็จ  

---

## 3. พลังแฝง (latent) ต่ออาชีพ

- แต่ละอาชีพมี `latent_growth` ใน `data/occupations/`  
- ลงทุน 1 แต้ม “โจมตี” ของนักรบ ≠ นักเวทย์  
- noise เล็กจาก `latent_seed` ต่อตัวละคร  
- combat: atk/crit/speed → ดาเมจ · magic → สกิล · defense → ลดดาเมจ soft  

---

## 4. พร (Blessing) ตอนเลเวลอัพ

1. ได้แต้มสถานะ  
2. roll พร — โอกาสต่ำ (`data/blessings/blessings.yaml`)  
3. passive เงียบ / เลือก 2–3 ทาง flavor / พรฉุกเฉินตอน HP ต่ำ  

หายาก · รู้สึกแรงชั่วคราว · ไม่พัง endless scaling  

---

## 5. อาชีพ · ขั้น · ทางแยก

```text
vagabond
    │  soft OR หลายชุด (เลเวล·kill·stat·personality·เควส·โชค)
    ▼
เมนู C เมื่อ qualifying
    ├─ อาชีพจาก data (นักรบ / เวทย์ / …)
    └─ เส้นทางแปลก (unit / flag)
```

**ขั้นอาชีพ (ranks[] ซ่อน):** min_level / min_stat_* / kills / boss ฯลฯ  
ครบ → ฉายา + สกิลสาย — ผู้เล่นไม่เห็น checklist

## 6. ห้องสมุด (**L**) · Unit

| ระบบ | กฎสั้น |
|------|--------|
| ห้องสมุด | access ซ่อนหรือกุญแจหีบ · ความรู้สุ่ม 1 ชิ้น/ครั้ง · คูลดาวน์ |
| อาชีพหลัก (offer) | ระบบ**เสนอ**ทาง · ผู้เล่น **y=รับ / n=ปฏิเสธถาวร** · ปฏิเสธแล้วอาชีพนั้นหาย · อาชีพอื่นยังเสนอได้ · เงื่อนไขซ่อน |
| ลงแต้ม | **อิสระ** — ไม่บังคับตามสาย · latent growth อาชีพแค่แผ่ว |
| อาชีพลับ (Unit) | **33 ชนิด** · 1/ตัว · 1/โลก · โอกาสต่ำมาก |
| Unit HSR | **joke** ตรงสไตล์→โหด · **broken** ไม่ตรง→แทบไร้ค่า · แต้ม+สไตล์ซ่อน · soft「เข้าทาง/พอใช้/ผิดทาง」 |

## 8. Anti-spoiler

- ห้ามโชว์ % คริ / หลบ / โชคดิบใน UI ปกติ  
- soft label: “รู้สึกเร็วขึ้น” / “ลมหายใจลึกขึ้น”  
- พร: ชื่อ flavor ไม่บอกตัวเลข  
- ตัวเลขบาลานซ์ = YAML ไม่ copy ตารางยาวลงนี่
