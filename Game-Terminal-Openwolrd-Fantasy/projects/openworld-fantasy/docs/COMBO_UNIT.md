# Combo 2.0 · Unit mastery · เกียร์เสริม (1.6.0)

## Combo 2.0
| เลเวล | สูงสุดขั้น |
|------:|----------:|
| 1–9 | 2 |
| 10–19 | 3 |
| 20–34 | 4 |
| 35–49 | 5 |
| 50+ | 6 |

- มานา × `[1.0, 1.12, 1.28, 1.48, 1.75, 2.10]`
- ดาเมจ power × ตามความยาว
- **มานาไม่พอ = ใช้ไม่ได้** (preview ก่อนยิง)
- Unit mastery ≥ 3 → +1 ความยาว (cap 6)
- fusion ~21 สูตร (รวม 3 ธาตุ)

## ปาร์ตี้ช่วยตี
- หลังโจมตีผู้เล่น: สมาชิกมีโอกาสช่วยดาเมจ (ผูก bond/ชนิด)
- เรียกใช้ (ปุ่ม 5) ยังเป็นบัฟหนักชั่วคราว

## Unit
- **claim** ต่อ `world_id`: ไฟล์ `saves/<world>/unit_claims.json`
- ชนิดละ 1 เจ้าของในโลก
- **mastery 0–5**: เริ่มอ่อน (power×0.55, mana×1.35) → ตื่นเต็ม
- เกจซ่อน: ใช้ unit skill / คอมโบยาว / โจมตี

## เครื่องประดับ
- ช่อง `accessory`: แหวน/สร้อย/มงกุฎ
- สวม/อัปได้เหมือนเกียร์อื่น
