# Vision: สมาธิ · ความฉลาด · เวท แฝง กับ Multi-Skill (คอมโบ)

**สถานะ:** **CM1–CM5 ship แล้ว (1.45–1.47)** · แผน Combo Mind ปิดก้อน alpha  
**คู่กับ:** [`combo.py`](../game/domain/combo.py) · [`intelligence.py`](../game/domain/intelligence.py) · [`SKILL_SLOT_RANK_VISION.md`](SKILL_SLOT_RANK_VISION.md)  
**หลัก soft:** สูตรซ่อน · วิธีเติบโตไม่บอกตรง · โชว์แค่ความรู้สึก/แบนด์

## 0. สิ่งที่มีอยู่ vs ที่คุณต้องการ

| แนวคิดของคุณ | ของปัจจุบัน (1.44) | ช่องว่าง |
|--------------|-------------------|----------|
| **สมาธิ (แฝง)** กำหนดใช้หลายสกิล | คอมโบยาวตาม **เลเวล** (+ unit mastery) | ยังไม่ผูกสมาธิ |
| ผู้เล่น**ไม่รู้ค่า** / ไม่รู้วิธีได้ | — | ต้องสร้าง latent ใหม่ |
| **ความฉลาด** โชว์ แต่เพิ่มเองไม่ได้ | `stats_alloc.intelligence` **ลงแต้ม P ได้** · บาร์สติ `intel_current/max` · `power_intel` | ต้องแยก “โชว์” กับ “ลงแต้ม” |
| สองค่าสูง → คอมโบยาว + **มานาแพง** | โซ่ยาวมี `mana_mult` ชันอยู่แล้ว | ยังไม่ผูก mind stats |
| **เวทแฝง** ลดมานาคอมโบ | `power_mag` มีแล้ว (ดาเมจเวท) · ยังไม่ลด MP โซ่ | ต่อ `mana_mult` soft |

**ทิศออกแบบ:** ไม่ทิ้ง combo 2.0 — **เปลี่ยน “เพดานขั้น + ภาษีมานา” ให้มาจากสมาธิ/ฉลาด/เวท** แทนเลเวลอย่างเดียว (เลเวลยัง soft cap)

## 1. นิยาม 3 ค่า (ล็อกความหมาย)

### 1.1 สมาธิ — `focus_latent` (ซ่อน 100%)

| | |
|--|--|
| **บทบาท** | ความนิ่งของจิตตอนเรียงท่า — **ขยายจำนวนขั้นคอมโบ** |
| **UI** | **ไม่โชว์ตัวเลข** · soft เท่านั้น เช่น 「จิตนิ่งพอเรียงท่า」/「จิตกระจัด」 |
| **ผู้เล่นรู้วิธีได้ไหม** | **ไม่** — หลายทางซ่อน (ด้านล่าง) |
| **เก็บ** | `player["focus_latent"]` float ซ่อน (หรือ derive ตอน recompute ไม่ persist ก็ได้) |

### 1.2 ความฉลาด — `mind_intellect` (โชว์ soft / แบนด์)

| | |
|--|--|
| **บทบาท** | ความเข้าใจโครงท่า — **ขยายขั้นคอมโบ** + **เพิ่มภาษีมานาโซ่** (คิดเป็นระบบ) |
| **UI** | โชว์ได้แบบ **แบนด์** ไม่โชว์สูตร: 「ความฉลาด: เลือน / พอใช้ / คม / แจ่ม」 หรือเลข soft ที่ **ไม่ใช่แต้ม P** |
| **เพิ่มเองไหม** | **ห้ามลงแต้ม P ตรงๆ เป็นทางหลัก** (ออกแบบใหม่ด้านล่าง) |
| **เก็บ** | ใช้/derive จาก `power_intel` + learn + path · แยกจาก `intel_current` (สติใช้เร่ง ATB) |

> **แยกคำในเกม (สำคัญ):**  
> - **ความฉลาด (mind)** = ฉลาดเรื่องท่า/คอมโบ (โชว์แบนด์)  
> - **สติ (intel pool)** = ทรัพยากรใช้เร่งจังหวะ/ทางเลือก ★ (`intel_current`) — คนละชั้น  

### 1.3 เวทแฝง — `power_mag` (ซ่อนจาก UI หลัก)

| | |
|--|--|
| **บทบาท** | ควบคุมการไหลมานา — **ลดตัวคูณมานาคอมโบ** (ไม่ลดดาเมจสกิลตรงในสูตรนี้) |
| **UI** | **ไม่โชว์** · soft หลังร่าย เช่น 「มานาไหลนิ่ม」 |
| **แหล่ง** | ลง magic / เกียร์เวท / อาชีพ mage / unit — ผู้เล่นอาจลง magic รู้ว่าแรงขึ้น แต่**ไม่รู้ว่าลด MP โซ่** |

## 2. สูตรคอมโบ (แนว — ซ่อนจากผู้เล่น)

### 2.1 จำนวนขั้นสูงสุด

```text
base = f(level)                    # ตารางเดิม 2..6 soft
+ steps_from_focus(focus_latent)   # 0..+2
+ steps_from_intellect(mind)       # 0..+2
+ unit_mastery_bonus               # 0..+1 เหมือนเดิม
hard_cap = 6 (หรือ 7 ถ้า unit)
```

**ตัวอย่างแบนด์ (ซ่อน):**

| focus_latent | +ขั้น |
|--------------|------|
| < 8 | 0 |
| 8–19 | +1 |
| ≥ 20 | +2 |

| mind_intellect | +ขั้น |
|----------------|------|
| < 6 | 0 |
| 6–14 | +1 |
| ≥ 15 | +2 |

ผล: สมาธิ+ฉลาดสูง → เรียงได้ยาว (สูงสุดยังมี cap กันพัง)

### 2.2 มานาคอมโบ

เดิม: `total_mana = sum(cost) * mana_mult[n-1]`

ใหม่:

```text
mult = mana_mult[n-1]                    # ตารางตามความยาว
     * intellect_mana_tax(mind)          # ฉลาดสูง → ภาษีขึ้น (คิดลึก = แพง)
     * focus_mana_tax(focus)             # สมาธิสูง → ภาษีนิด (คุมหลายท่าหนัก)
     / mag_mana_relief(power_mag)        # เวทแฝง → หารลง (ไหลนิ่ม)
clamp mult ∈ [0.75, 2.8]
```

**ทิศทางตามคุณ:**  
- ฉลาด + สมาธิ **สูง → ใช้หลายท่าได้ + มานาหนัก**  
- เวทแฝงสูง → **มานาหมดช้า** (ลดภาษีโซ่ ไม่ทำให้ free spam ถ้า n ยาวมาก)

**สมดุลสำคัญ:**  
อย่าให้เวทยกเลิกภาษีทั้งก้อน — relief cap เช่น ลดได้สูงสุด ~25–35% ของ mult หลัง tax

### 2.3 Soft UI ตอนไฟต์

แทนตัวเลข:

- 「คอมโบสูงสุด **3** ขั้น」← ตัวเลขขั้นโชว์ได้ (รู้เพดานเล่น)  
- ไม่โชว์ “เพราะสมาธิ 12”  
- ถ้า MP ไม่พอ: 「โซ่นี้หนักเกินไปสำหรับมานา/จิตตอนนี้」  
- หลังร่ายโซ่ยาว: 「จิตล้า」 optional ลด `focus_latent` ชั่วคราว / กินสติ pool

## 3. วิธีได้ค่า (ซ่อน — ออกแบบหลายทาง)

### 3.1 สมาธิ `focus_latent` (ไม่บอกผู้เล่น)

| ทาง | แนว | soft ใบ้ (ไม่ชี้สูตร) |
|-----|------|----------------------|
| A | พัก / rest / วัด | 「จิตค่อยสงบ」 |
| B | ชนะไฟต์โดย**ไม่หนี** · ไม่ spam ยา | — |
| C | ใช้สกิล**ทีละท่า** บ่อย (ฝึกนิ่ง) ก่อนโซ่ยาว | 「มือคุ้นก่อนจิตนิ่ง」 |
| D | บัฟ focus / ชา / ธูป | ของมีอยู่ |
| E | อาชีพ priest/monk-like / blessing | path soft |
| F | แพ้/ขวัญแตก → สมาธิลด soft | needs morale |
| G | โซ่ยาวเกินกำลัง → ล้า (ลดชั่วคราว) | 「เรียงมากไป จิตกระจัด」 |

**ห้าม:** เมนู “อัปสมาธิ +1”

### 3.2 ความฉลาด (โชว์แบนด์ · ไม่ลง P ตรง)

**เปลี่ยนจากปัจจุบัน:**  
เอา `intelligence` ออกจากทางหลักของเมนู **P** (หรือคงไว้แต่ rename เป็น “สมาธิฝึก” คนละอย่าง — แนะนำ **ถอด intelligence ออกจาก allocate ฟรี**)

ทางเพิ่ม `mind_intellect` / `power_intel` ซ่อน:

| ทาง | แนว |
|-----|------|
| A | เรียนสกิลใหม่ / อ่านห้องสมุด |
| B | สำเร็จเควส “คิด” · master teach |
| C | ชนะด้วย fusion / คอมโบธาตุตรง |
| D | ใช้สติ (intel pool) บ่อยแบบมีสติ — soft |
| E | เกียร์ “แว่น/คัมภีร์” latent |
| F | Unit / occupation mage-scholar |
| G | เลเวลขึ้น soft นิด (น้อย) |

**UI โชว์:** แบนด์จากค่า derive เท่านั้น · หลังเหตุการณ์ 「รู้สึกคิดคมขึ้น」ไม่โชว์ +3

### 3.3 เวทแฝง `power_mag`

| ทาง | แนว | ผู้เล่นรู้ไหม |
|-----|------|--------------|
| ลงแต้ม **magic** | รู้ว่าเวทแรง | **ไม่รู้**ว่าลด MP คอมโบ |
| เกียร์/การ์ดเวท | รู้ ATK เวท soft | ไม่รู้ MP relief |
| อาชีพ mage | path | soft |

## 4. ความสัมพันธ์กับระบบอื่น

| ระบบ | เชื่อม |
|------|--------|
| **สติ pool** (`intel_current`) | คนละชั้น · โซ่ยาว optional เสียสติ 0–1 · ไม่ใช้สติเป็น “ขั้นคอมโบ” หลัก |
| **Skill Rank** | scale power/MP ต่อท่า **ก่อน** รวมโซ่ · mind ไม่แทน rank |
| **Buff gate** | ยัง 1 บัฟ/แอคชัน — mind สูง**ไม่**ให้บัฟซ้อน |
| **Unit HSR** | mastery +1 ขั้นคงได้ · style match soft กับ focus gain |
| **Needs** | หิว/ล้า/ขวัญต่ำ → focus_latent ชั่วคราวลด · คอมโบสั้นลง soft |
| **MI มอน** | ไม่เกี่ยวตรง · optional ฉลาดสูง soft talk |

## 5. ความเสี่ยงและ mitigation

| ความเสี่ยง | แก้ |
|------------|-----|
| Mage เวทสูง + ฉลาดสูง = โซ่ 6 ฟรี | relief เวทมี cap · tax ฉลาดชัน · hard_cap 6 |
| ผู้เล่นงงทำไมคอมโบเหลือ 2 | soft ใบ้จิต/เลเวล · ไม่โชว์ตัวเลข latent |
| ถอด intelligence จาก P แล้ว build เก่าพัง | migrate: แต้ม int เก่าย้ายเข้า mind_intellect ครั้งเดียว |
| สมาธิ grind พิษ | multi-path · soft ceiling ต่อเลเวล · diminishing |
| ซ้ำกับสติ UI | ชื่อคนละคำ · แถบคนละที่ |

## 8. สรุปความเห็นออกแบบ

| ข้อคุณเสนอ | ความเห็น |
|------------|----------|
| สมาธิซ่อน คุม multi-skill | **ดีมาก** — เติมช่องว่างที่ตอนนี้ใช้แค่เลเวล |
| ฉลาดโชว์ เพิ่มเองไม่ได้ | **ดี** กับ soft RPG — ต้อง **แยกจากสติ pool และเลิกลง P ตรง** |
| สองค่าสูง = หลายท่า + มานาแพง | **สมเหตุผล** — กัน “ฉลาด = free power” |
| เวทแฝงลดมานา | **ดี** — ให้สายเวท identity โดยไม่โชว์สูตร · ต้องมี **cap** |

**ไม่ชน** กับ Skill Rank / buff gate ถ้าคง “ขั้น + ภาษี MP” คนละชั้นกับ rank ต่อท่า
