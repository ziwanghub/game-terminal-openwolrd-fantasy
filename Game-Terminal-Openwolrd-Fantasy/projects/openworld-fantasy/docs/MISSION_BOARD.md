# ประกาศภารกิจ + ภาษีตลาด

**เวอร์ชัน:** 1.13.1-alpha

## วงจรเศรษฐกิจ

```text
ขายของในตลาดกลาง
  → หักค่าธรรมเนียม (ภาษี)
  → เข้า tax_fund ของโลก
  → ใช้เป็นค่าจ้างประกาศภารกิจ (wage_cost)
  → ผู้เล่นรับงาน · เคลียร์ · ได้รางวัลตามแรงก์
```

## แรงก์

```text
F < E < D < C < B < A < S < SS < SSS
```

- เริ่มต้น: **F**
- รับงานได้: แรงก์งาน ≤ แรงก์ผู้เล่น
- เลื่อนแรงก์: **รับงาน + เคลียร์บ่อย** (สะสม mission_rank_xp — เกณฑ์ไม่เปิดหมด)

## งานพิเศษ (★)

- เงื่อนไขรับ **ซ่อน** (เลเวล/คิล/สติ/โชค/ประวัติ)
- ผู้เล่นต้องประเมินตัวเอง — ระบบไม่บอกทำไมถึงรับได้/ไม่ได้

## UI

```text
กระเป๋า → J ประกาศภารกิจ
ตลาด → แสดงกองทุนภาษี
```
