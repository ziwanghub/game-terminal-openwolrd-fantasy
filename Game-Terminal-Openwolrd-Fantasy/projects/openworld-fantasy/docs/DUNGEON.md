# ระบบดันเจียน

## หลักการ
- แต่ละพื้นที่อาจมี **ปากดันเจียน** โผล่ใน “สิ่งที่สังเกตเห็น”
- **ความยากเป็นตัวเลขซ่อน** — ผู้เล่นเห็นแค่คำคลุมเครือจนกว่าจะเคยเข้า/เคลียร์
- เข้าแล้ว **ทางออกล็อก** จนกว่า:
  1. เคลียร์บอสดันเจียน หรือ
  2. ใช้ของพิเศษที่**มีอยู่แล้วตอนเข้า** (วิธีได้มาไม่บอกในเกม)

## การเล่น
1. เจอ `[dungeon]` ในพื้นที่ → เข้าหา (3)
2. เตรียมปาร์ตี้ (Y) ถ้าต้องการ → ยืนยันเข้า
3. ในดันเจียน:
   - สำรวจชั้น (สู้)
   - ลงลึก
   - ท้าบอส (ชั้นในสุด)
   - พยายามออก / ใช้ของหนี
4. เคลียร์บอส → ทางออกเปิด

## ถ้าออกไม่ได้ / ล้มเหลว
- พยายามออกโดยไม่มีของ / ของไม่สำเร็จ → **เสียทรัพยากร** (เงิน + โอกาสเสียของ)
- สลบในดันเจียน → soft death + ถูกเหวี่ยงออก + เสียของ

## ของหนี (ซ่อนวิธีใช้)
| id | ชื่อ | หมายเหตุ |
|----|------|----------|
| dungeon_thread | ด้ายกลับทาง | โอกาสหนีปานกลาง |
| void_key_shard | เศษกุญแจสุญญะ | โอกาสสูง |
| blessed_charm | เครื่องรางสวรรค์ | โอกาสต่ำ (ใช้ได้หลายอย่าง) |

มีในกระเป๋า**ตอนก้าวเข้า** ระบบจะจำ — ใช้ตอนพยายามออก

## ความรู้
- เข้าหลายครั้ง → ใบ้ความอันตรายชัดขึ้น
- เคลียร์สำเร็จ → ข้อความ “accurate”

## รางวัลเคลียร์ (1.8.1)
- หลังบอสล้ม: เงิน / XP / ไอเทมเฉพาะดันเจียน (ครั้งเดียวต่อรอบ)
- แต่ละถ้ำมี `rewards:` ใน data (ของหนี/วัสดุ/เกียร์)

## ภูมิชั้นสุ่ม
- ทุกชั้นได้ layout tag: ทางคด · โถง · ทางแคบ · ถล่ม · แท่น · แอ่ง · ผลึก · สุญญะ
- สำรวจชั้น = สุ่มเหตุการณ์: ซุ่ม / ลาดตระเวน / ว่าง / หีบ / พัก / กับดัก / ลาง

## จำกัดเวลา (ซ่อนตัวเลข)
- แต่ละแอคชันสำคัญในดันเจียนใช้ 1 หน่วยเวลาภายใน
- ใกล้หมด: ข้อความกดดัน · ศัตรูแรงขึ้นเล็กน้อย
- หมด: มิติดันเจียนยุบ → เหวี่ยงออก + เสียทรัพยากร
