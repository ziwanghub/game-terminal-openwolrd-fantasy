# ระบบนิสัย (Personality)

## หลักการ
- นิสัยถูกหล่อหลอมจาก**การเลือก**และ**สไตล์การเล่น** ไม่ใช่เมนูตั้งค่าตรงๆ
- ผู้เล่นเห็นแค่ป้ายคลุมเครือ (เมื่อแกนนิสัยเด่นพอ)
- ค่าจริง -100..100 ซ่อน · สูตรความเข้ากันซ่อน
- **แต้มนิสัย (personality_points)** ผู้เล่นลงทุนเองได้ — แต่**วิธีได้แต้มซ่อน**

## แกนนิสัย (8)
kindness, courage, honor, greed, curiosity, aggression, loyalty, caution

## แต้มนิสัย (ลงทุนเอง)
- เมนู **N** → ดูฉลาก + ลงทุนแต้มไปทางขั้ว「สูง」ของแกน
- ลงทุนแล้วเห็นแค่จำนวนที่ใส่ ไม่เห็นตัวเลขแกนจริง
- วิธีได้แต้ม: **ต้องสังเกตจากการเล่น** หรือใบ้จากห้องสมุด (ไม่ครบ)
- เงื่อนไขเกณฑ์อยู่ใน `data/personality/point_grants.yaml` (engine only)

### แนวคิดการได้แต้ม (ดีไซน์ — อย่าโชว์ใน UI)
- การเข้าหาแบบสุภาพ/ของขวัญ/ช่วยเหลือ สะสม
- ชนะการต่อสู้ / สตรีคไม่หนี / คอมโบ
- สำรวจ ห้องสมุด เควส เลเวล บอส พบผู้เล่นอื่น
- บางอันครั้งเดียว · บางอันช้าๆ ซ้ำได้
- ได้แต้มแล้วข้อความคลุมเครือ: 「ช่องว่างในใจเปิดขึ้น」— ไม่บอกเพราะอะไร

## ห้องสมุด — เศษข้อมูลนิสัย
- แยกพูลจาก knowledge entries ปกติ (`data/library/personality_tips.yaml`)
- **fragment**: ข้อมูลบางส่วน ต้องเดา
- **rumor**: อาจผิด / หลอก
- **complete**: ครบถ้วนและถูกต้อง — **weight ต่ำมาก** (โชคดี)
- ครั้งเดียวไม่ unlock ทุกอย่าง · อ่านแล้วจำ id กันซ้ำ (pool หมดแล้วสุ่มซ้ำได้)

## แหล่งที่ทำให้แกนนิสัยเปลี่ยน (passive)
- ท่าทางเข้าหา NPC / ผู้เล่น (polite, gift, challenge, threaten, cautious, aid)
- การต่อสู้: โจมตี / กัน / หนี / คอมโบ / ชนะ / ตาย
- สำรวจ พัก หีบ ร้าน อาจารย์ ห้องสมุด เควส
- **บวก** การลงทุนแต้มนิสัยจากเมนู N

## ผลต่อความสัมพันธ์
### ผู้เล่นอื่น (echo)
- ความคล้ายเวกเตอร์นิสัย + clash เมตตา/ก้าวร้าว
- รวมเข้า affinity เดิม (path, เกียร์, สเตตัส, ท่าทาง)

### NPC (archetype ซ่อน)
- merchant / master / bandit / pilgrim / scholar / wanderer
- prefers / dislikes ต่อแกนนิสัย
- ผล: โอกาสเพื่อน-ศัตรู, ส่วน/แพงร้าน, สอนสกิล, ใบ้ห้องสมุด, ซุ่ม

## UI
- **N** = แผงนิสัยคลุมเครือ + ลงทุนแต้ม
- **L** = ห้องสมุด (อาจสุ่มได้เศษบันทึกนิสัย)
- ไม่มีแถบตัวเลขนิสัย · ไม่มีรายการเงื่อนไขได้แต้มในเกม
