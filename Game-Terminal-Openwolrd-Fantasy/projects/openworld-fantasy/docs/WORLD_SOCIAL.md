# ระบบโลกอิสระ · อันดับ · เจอผู้เล่น

## โลก
- มีชื่อ · ระดับความยาก · modifiers
- อิสระ: เซฟ / ประวัติ / social_memory / อันดับ แยกตาม world_id
- เข้าโลก → เล่นต่อเซฟล่าสุดของโลกนั้นได้

## อันดับ (เมนู 3)
แสดง: อันดับ, ชื่อ, สายอาชีพ, ฉายาชื่อเสียง  
**ไม่แสดง:** เลเวล, HP, ATK, ค่าพลัง  
คะแนนจัดลำดับคำนวณซ่อน (kills/boss/quests/library/unit/…)

## เจอผู้เล่นในโลก
- ดึงจากเซฟอื่นในโลกเดียวกัน (player echo)
- ใช้ข้อมูลจริง: อาชีพ, สเตตัสลงทุน, เกียร์, การ์ด, สกิล
- ผู้เล่นเลือกท่าทางเข้าหา (ป้ายอย่างเดียว)
- **affinity ซ่อน** → เพื่อน (ร่วมสู้/บัฟ) หรือ ศัตรู (PvP กับสเตตัสจริง)

## ตัวแปรซ่อนที่มีผลเป็นเพื่อน/ศัตรู (ผู้เล่นไม่เห็น)
ดู `data/worlds/social.yaml`
- path อาชีพเดียวกัน / ใกล้ / คู่ปรับ
- ความคล้ายเวกเตอร์แต้มสเตตัส
- แท็กเกียร์ซ้อน
- Unit ตรง/ไม่ตรง
- ช่องว่างกดดัน
- ประวัติ friend/rival กับ id นั้น
- ช่องว่างคะแนนอันดับซ่อน
- ท่าทางเข้าหา (polite/gift/challenge/threaten/cautious/aid)
- noise สุ่ม
