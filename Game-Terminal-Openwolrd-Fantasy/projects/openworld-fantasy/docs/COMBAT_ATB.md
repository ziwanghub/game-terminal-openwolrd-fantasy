# ระบบแท่งจังหวะต่อสู้ (ATB)

**สถานะ:** implement 1.12.1  
**หลัก:** ทุกฝ่ายต้องรอแท่งสเตตัสเต็มก่อนเลือกคำสั่ง · ความเร็วไม่เท่ากัน · **ผู้เล่นไม่เห็นสูตร**

---

## การทำงาน

```text
เริ่มไฟต์ → init gauge (ซุ่ม: ศัตรูเกือบเต็ม)
loop:
  เติมแท่งตาม rate ซ่อน จนมีฝ่าย ≥ 100
  แสดงแท่ง soft: [████░░░░░░] พร้อม / เกือบเต็ม / ช้า
  ฝ่ายที่เต็ม → เลือกคำสั่ง (ผู้เล่น) หรือโจมตี (มอน)
  ใช้แล้วแท่งถูกรีเซ็ต (เหลือเศษ overfill เล็กน้อย)
```

---

## ปัจจัย rate (ซ่อน — ไม่โชว์ตัวเลข)

| ฝ่าย | ดึงจาก |
|------|--------|
| ผู้เล่น | power_spd · ลงทุนความเร็ว · กดดัน · พร · luck_score · สถานะ (freeze/slow/haste) · อาชีพผ่าน latent |
| มอน | เลเวล · atk base · boss ช้าลงเล็กน้อย · rarity · สถานะ |

ผู้เล่นเห็นแค่แท่งเต็มช้า/เร็ว — ไม่มี % หรือสูตร
