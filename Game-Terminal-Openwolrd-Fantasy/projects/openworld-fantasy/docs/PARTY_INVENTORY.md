# ปาร์ตี้ · กระเป๋า · ดรอป · อัปเกรด (1.5.0-alpha)

## ปาร์ตี้ (สูงสุด 3)
- ชนิด: ผู้เล่น · ภูต · สัตว์อสูร · เทพสวรรค์ · เทพมาร · สัตว์สวรรค์ · สัตว์นรก · อื่น
- **ต้องได้รับการยอมรับ** ถึงเข้าทีม (เงื่อนไขซ่อน)
- ผู้เล่นอื่น: มิตรภาพ → ชวนเข้าปาร์ตี้ (consent roll)
- สิ่งมีชีวิต: สำรวจมีโอกาสเงาเข้าใกล้
- **เรียกใช้** ในไฟต์ (ปุ่ม 5) มีต้นทุน MP/เงิน ตามสมาชิก
- พาสซีฟเล็กน้อยตลอด · เรียกใช้ = บัฟแรงชั่วคราว
- เมนู **Y**

## กระเป๋า
- ความจุ soft cap 40 (inventory + cards)
- เมนู 5 → 7 ดูรายการ · เลือกดู **คุณสมบัติ + วิธีใช้**
- เกียร์แสดง ATK/HP/MP ที่เสริมจากชิ้นสวม

## ดรอป
- ชนะไฟต์ → รายการของหล่น → **ผู้เล่นเลือกเก็บ** (comma / 0=ทิ้ง)

## อัปเกรด
- มี **อัตราสำเร็จ** ลดลงเมื่อ + สูง
- ล้มเหลว: วัสดุ/เงินสลาย ขั้นไม่ขึ้น
- UI ไม่บอกจำนวนวัสดุชัด — ใบ้คลุมเครือ + ระดับความเสี่ยง
