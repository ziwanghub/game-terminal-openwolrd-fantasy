# RD-11: Save / Offline Requirements  
*(แทน PWA — ผลิตภัณฑ์ไม่ใช่เว็บแอป)*

**สถานะ:** Draft  
**ชื่อไฟล์เดิม:** pwa-requirements — ความหมายใหม่: ออฟไลน์ + เซฟท้องถิ่น

---

## 1. วัตถุประสงค์

เกมต้องเล่น **ออฟไลน์เต็มรูปแบบ** และเก็บความคืบหน้าบนเครื่องผู้เล่น

## 2. ข้อกำหนด

| ID | ข้อกำหนด |
|----|----------|
| OFF-01 | ไม่บังคับเครือข่ายตอนเล่น/เซฟ/โหลด |
| OFF-02 | เซฟอยู่ใต้ `saves/` ของโปรเจกต์ (หรือ path ที่กำหนด) |
| OFF-03 | ผู้เล่นเลือกตัวละครจากเซฟที่มี |
| OFF-04 | ไม่ sync คลาวด์ใน v1 |
| OFF-05 | เอกสารบอกวิธีสำรองโฟลเดอร์ saves |
