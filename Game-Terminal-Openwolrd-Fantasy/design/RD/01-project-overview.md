# RD-01: ภาพรวมโครงการ

**สถานะ:** Draft  
**ผลิตภัณฑ์:** Open World Fantasy (Terminal Text RPG)  
**รหัสโปรเจกต์:** `projects/openworld-fantasy`

---

## 1. วัตถุประสงค์

สร้างเกมแฟนตาซีแนว **open world + open skill** ที่เล่นได้บน **Terminal แบบข้อความล้วน**  
ผู้เล่นสำรวจพื้นที่ เรียนรู้ระบบผ่านประสบการณ์ (ไม่ถูกบอกสูตรทั้งหมด) สู้ด้วยสกิล/คอมโบ/การป้องกันแบบแพ้ทาง เติบโตด้วยเลเวล เกียร์ และการ์ด — โดยคอนเทนต์แยกไฟล์ขยายได้

## 2. วิสัยทัศน์ผู้เล่น

> RPG ข้อความที่โลกเปิดกว้างจากข้อมูล, ความไม่รู้สร้างความตื่นเต้น, ทักษะการอ่านเกมสำคัญกว่าเลเวลล้วนๆ, และจังหวะพิเศษโดดเด่นด้วยข้อความ/ASCII โดยไม่มีกราฟิกจริง

## 4. นอกขอบเขต (Out of Scope) — v1

| ไม่รวม | เหตุผล |
|--------|--------|
| กราฟิก 2D/3D / เอนจินเกมภาพ | เจตนา text-only |
| มัลติเพลเยอร์เรียลไทม์ | ใช้จำลองผู้เล่นอื่นก่อน |
| เว็บไซต์ SEO / PWA / โฆษณา | ไม่ใช่ผลิตภัณฑ์นี้ |
| ระบบชำระเงิน | ไม่มีใน v1 |
| Mobile native app | เล่นผ่าน Terminal/SSH พอ |

## 6. ตัวชี้วัดความสำเร็จ (ระดับผลิตภัณฑ์)

1. ผู้เล่นสร้างตัว เล่นสำรวจ สู้ เซฟ แล้วโหลดต่อได้  
2. เพิ่มพื้นที่/มอน/สกิลใหม่ด้วยไฟล์ data โดยไม่ rewrite เอนจินหลัก  
3. ผู้เล่นอย่างน้อยค้นพบ 1 ปฏิกิริยาธาตุหรือท่ากันด้วยตัวเอง (journal)  
4. เหตุการณ์ของเทพแสดง Spotlight ASCII ชัดเจน  
5. รันบน Terminal มาตรฐานและ SSH ได้
